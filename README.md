# CHK-1772477091912-6098
IoT-based Air Quality Monitoring System using ESP32 that measures carbon monoxide, harmful gases, dust concentration, temperature, and humidity using MQ7, MQ135, GP2Y dust sensor, and DHT11 sensors. It also provides GPS coordinates using the NEO-6M GPS module. Data is visualized using AirWatch and VayuMitra being the mediator for citizen and gov.
